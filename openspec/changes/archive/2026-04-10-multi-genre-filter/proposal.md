## Why

当前风格筛选功能只支持单选，用户无法同时筛选多个风格（如同时筛选 Rock 和 Jazz）。这限制了用户进行精确查找跨风格专辑的能力。此外，专辑列表中的风格标签仅供展示，无法直接点击进行筛选，降低了交互效率。

## What Changes

- 将风格筛选从单选下拉框改为支持多选的标签输入组件（带自动完成）
- 多风格筛选使用 AND 逻辑：专辑必须同时包含所有选中的风格才会显示
- 在工具栏显示已选中的风格标签，可点击移除或一键清除全部
- 专辑列表和详情中的风格标签变为可点击，点击可切换筛选状态
- 已选中的风格标签使用不同样式高亮显示

## Capabilities

### New Capabilities
- `multi-genre-filter`: 多风格筛选功能，支持同时选择多个风格进行 AND 逻辑筛选，包括标签输入组件、已选标签展示、可点击的风格标签交互

## Impact

- **前端组件**: `App.vue` 需要重构风格筛选器 UI 和风格标签组件
- **筛选逻辑**: 后端 API 需要支持多风格 AND 筛选查询
- **样式**: 需要添加新的 CSS 样式用于标签输入组件和已选状态高亮
